Reliability:

Setup

The setup for the crane mission is important, but it does not have to be precise. There is a balck line that we can allign the blue peg with.

Run

We use the balck line to align the robot so that the attachment is working well.

Mechanical

We have an attachment designed that pushes the block and activates the blue levers at the righ time. The drive must not be fast into the crane. The crane block must not swing wne we start.

Mission Order

To avoid the swinging of the line (by other teams moving the table), this is our first mission.
